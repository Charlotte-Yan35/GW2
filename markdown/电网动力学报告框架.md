## 3. 引言 (Introduction) 【2-3页】

### 3.1 背景与动机
- **传统电网 vs 现代电网**：
  - 传统：少量大型集中式发电机
  - 现代：分布式可再生能源（光伏、风电），双向功率流
- **电网运行的两个关键约束**：
  1. 频率同步（UK/欧洲为50 Hz）
  2. 功率流在工程容限内
- **挑战**：理解分布式可再生能源整合如何影响不同尺度的电网功能（微电网 → 国家配电网）

### 3.2 文献综述
- **Swing/Kuramoto模型在电网稳定性中的应用**：
  - Filatrella等(2008)：电网的类Kuramoto模型分析
  - Nishikawa & Motter(2015)：电网模型的比较分析
  - Motter等(2013)：使用主稳定性函数的自发同步
- **级联失效模型**：
  - 拓扑/渗流方法：Motter & Lai(2003), Crucitti等(2004)
  - 动力学方法：Schäfer等(2018) - 动力学诱导的级联
- **研究空白**：
  - 现有研究主要关注大规模输电网络
  - 对具有真实供需变化的可再生能源主导微电网研究较少
  - 缺乏对小尺度电网拓扑结构高度可变性的设计原则

### 3.3 研究目标
1. **主要目标**：确定 P_k(t) 的变化、网络结构 A_kl 和模型参数如何共同影响电网稳定性
2. **具体目标**：
   - 分析网络拓扑(k, q)对临界耦合强度 κ_c 的影响
   - 研究节点组成（发电机/消费者/被动节点）对韧性 ρ 的影响
   - 探究级联失效动力学，识别脆弱区域
   - （扩展）探索异质阻尼 γ_i 的效应

## 4. 数学模型 (Mathematical Model) 【3-4页】

### 4.1 网络表示

#### 4.1.1 节点类型与功率注入 P_i
- **发电节点**：P_i > 0（功率源）
- **负荷节点**：P_i < 0（功率汇）
- **被动节点**：P_i ≈ 0（净零功率）
- **功率平衡约束**：∑_i P_i = 0（封闭系统）或通过PCC平衡

#### 4.1.2 网络拓扑：Watts-Strogatz模型
- **参数**：n（节点数）、k（平均度）、q（重连概率）
- **特殊情况**：
  - q = 0：规则环形格子（高聚类系数，长路径长度）
  - q = 1：随机网络（低聚类系数，短路径长度）
  - 0 < q < 1：小世界区域
- **可视化**：包含ring vs small-world网络示意图（n=30示例）
- **表格**：展示聚类系数和平均路径长度随q的变化

### 4.2 Swing方程动力学

#### 4.2.1 推导概要
- 物理基础：旋转机械模型
- 相位角 θ_i(t)：相对于电网参考频率 Ω 的偏差
- 功率平衡：惯性项 + 耗散项 + 传输功率 = 机械输入功率

#### 4.2.2 控制方程
$$\frac{d^2\theta_k}{dt^2} + \gamma\frac{d\theta_k}{dt} = P_k - \kappa\sum_{l=1}^{n} A_{kl}\sin(\theta_k - \theta_l), \quad k = 1...n$$

#### 4.2.3 参数定义与取值
| 参数 | 符号 | 物理含义 | 取值范围 | 来源 |
|------|------|----------|----------|------|
| 阻尼系数 | γ | 能量耗散率 | ~1 s⁻¹ | [29]，量级一假设 |
| 耦合强度 | κ | 传输容量 | V₀²B/(JΩ) | 由线路参数推导 |
| 网络邻接矩阵 | A_kl | 连接拓扑 | 0 或 1 | WS模型生成 |

**参数来源说明**（重要，避免扣分）：
- γ ≈ 1：基于典型发电机转动惯量J和阻尼系数D的量级估计
- κ 的物理上限：V₀²B ≈ 18.3 kW（基于V₀=240V，L=10mH）

#### 4.2.4 边上的功率流
$$f_{kl} = \kappa\sin(\theta_k - \theta_l)$$

### 4.3 稳定性与失效判据

#### 4.3.1 同步条件
- 稳态：P_i - κ∑_j A_ij sin(θ_i - θ_j) = 0 对所有 i 成立
- 临界耦合 κ_c：稳定同步运行所需的最小 κ

#### 4.3.2 失效机制
1. **过载失效**：边功率流 |f_kl| > α（容量阈值）
2. **去同步失效**：|dθ_i/dt| > 1 Hz（频率漂移超限）

## 5. 解析框架 (Analytical Framework) 【2-3页】

### 5.1 双节点系统分析（基准情形）
- **设置**：一个发电机(P)与一个消费者(-P)耦合
- **降阶系统**（令 Δθ = θ₁ - θ₂）：
  $$\frac{d(\Delta\theta)}{dt} = \omega$$
  $$\frac{d\omega}{dt} = 2P - \gamma\omega - 2\kappa\sin(\Delta\theta)$$
- **不动点**：(Δθ*, ω*) = (arcsin(P/κ), 0) 和 (π - arcsin(P/κ), 0)
- **鞍结分岔**：发生在 κ_c = P
- **分岔图**：展示稳定/不稳定分支的图

### 5.2 临界耦合 κ_c 分析
- **定义**：同步运行所需的最小耦合强度
- **计算方法**：数值延拓或二分法
- **依赖因素**：
  - 网络结构 (k, q)
  - 节点组成 (n_+, n_-, n_p)

### 5.3 韧性度量 ρ
- **定义**：ρ = α_c / α_* 其中 α_c 是使 S = 0.5 的临界容量
- **解释**：ρ 越低 → 韧性越强
- **分布特性**：ρ 在网络集合中服从对数正态分布

### 5.4 Simplex表示法
- **坐标**：(n_+, n_-, n_p) 或连续形式 (η_+, η_-, η_p)
- **可视化**：节点组成的三角形相空间
- **映射**：功率数据 → simplex轨迹

## 6. 数值方法 (Numerical Methods) 【2页】

### 6.1 ODE积分
- **求解器**：ode45/RK4 或刚性求解器(ode15s)
- **时间步长**：自适应，典型范围 [10⁻⁴, 10⁻²] s
- **停止准则**：
  - 稳态：max|dθ/dt| < ω_tol（如 10⁻⁴ Hz）
  - 残差：max|P_i - κ∑A_ij sin(θ_i-θ_j)| < res_tol

### 6.2 临界耦合计算
- 对 κ 进行二分搜索
- 收敛判据：积分是否达到稳态

### 6.3 级联失效算法
```
算法：动态级联
输入：网络 G = (V, E, P, α)
输出：存活边比例 S

1. 通过积分swing方程找到稳态 θ*
2. 移除最大功率流的边（触发失效）
3. 当网络未稳定时：
   a. 时间步进swing方程
   b. 移除过载边：|f_kl| > α
   c. 移除去同步节点：|ω_i| > 1 Hz
   d. 检查新的连通分量
4. 返回 S = 存活边数 / 总边数
```

### 6.4 集合平均
- 生成 N 个WS网络实现（N = 100-500）
- 每个实现随机化节点位置
- 报告指标的均值和标准差

## 7. 结果 (Results) 【5-6页】

### 7.1 网络拓扑对同步的影响

#### 7.1.1 κ_c vs (k, q)
- **图**：κ_c 均值关于度 k 和重连概率 q 的热力图
- **发现**：q 越高（越随机）→ κ_c 越低 → 越容易同步
- **解释**：更短的路径长度促进功率分配

#### 7.1.2 κ_c Simplex景观
- **图**：κ_c 投影到（发电机，消费者，被动节点）simplex上
- **发现**：当 n_+ ≈ n_-（组成平衡）时 κ_c 最小
- **对比**：q = 0 vs q = 0.1 vs q = 1

### 7.2 韧性分析

#### 7.2.1 ρ 分布
- **图**：格子网络 vs 小世界网络的 ρ 直方图
- **发现**：对数正态分布；小世界略微不那么韧性

#### 7.2.2 ρ Simplex景观
- **图**：不同 q 值下 ρ 投影到simplex
- **发现**：最具韧性的配置在底部中心（少被动节点，发电/消费平衡）
- **与 κ_c 对比**：两个指标倾向于相似区域

### 7.3 级联失效动力学

#### 7.3.1 S(α) 曲线
- **图**：存活比例 vs 归一化容量 α/α_*
- **发现**：S型转变；韧性较差的网络曲线更陡

#### 7.3.2 失效模式分解
- **图**：过载失效 vs 去同步失效的比例
- **发现**：当消费者 >> 发电机时，去同步失效占主导

#### 7.3.3 级联持续时间
- **图**：平均级联时间 T vs α/α_*
- **发现**：临界点处持续时间达峰值；灾难性级联很快

### 7.4 时变分析（数据驱动）⭐【核心创新部分】

#### 7.4.1 数据来源
- UK Power Networks：家庭用电（30分钟分辨率，2年）
- UK Power Networks：光伏发电（10分钟分辨率，1年）

#### 7.4.2 日功率曲线
- **图**：24小时内发电和消费的均值（季节对比）
- **发现**：消费早晚高峰；光伏中午高峰

#### 7.4.3 Simplex轨迹
- **图**：不同PV采用率的周轨迹（50% vs 100%）
- **图**：季节对比（冬季 vs 夏季）
- **发现**：PV采用率越高 → simplex中偏移越大
- **发现**：夏季轨迹到达左边缘（发电主导）

#### 7.4.4 鲁棒性的时间变化
- **图**：一周内 κ_c 时间序列
- **发现**：日振荡；电网平衡时短暂最小值
- **含义**：电网在最优区域停留时间很短

#### 7.4.5 失效位置分析
- **图**：级联失效在simplex上的发生点
- **发现**：失效集中在simplex边缘（组成不平衡）
- **图**：不同PV采用率下 α_c 的分布
- **发现**：夏季高PV → 双峰 α_c 分布 → 脆弱性增加

## 9. 讨论与结论 (Discussion and Conclusions) 【1-2页】

### 9.1 主要发现总结
1. **拓扑**：小世界网络（适度q）更易同步但韧性相当
2. **组成**：发电/消费平衡且少被动节点是最优配置
3. **时变性**：可再生能源整合导致韧性日振荡
4. **脆弱性**：夏季高PV采用率将电网推向临界容量

### 9.2 工程意义
- 小世界重连可能降低所需耦合容量
- 控制策略应以维持平衡组成为目标
- 电池部署策略：旨在使电网保持在simplex中心区域
- 冗余提高稳定性但有成本
