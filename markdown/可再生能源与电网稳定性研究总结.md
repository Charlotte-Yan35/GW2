# 可再生能源对电网稳定性和韧性的影响

**论文来源**: Smith et al., Science Advances 8, eabj6734 (2022)
**作者**: Oliver Smith, Oliver Cattell, Etienne Farcot, Reuben D. O'Dea, Keith I. Hopcraft
**机构**: 诺丁汉大学数学科学学院

---

## 一、研究背景与动机

### 1.1 能源转型的现状
- 英国可再生能源发电占比从2010年的6.9%上升至2019年的37.1%
- 可再生能源发电机（如光伏、风电）具有**低输出、间歇性**的特点
- 这种小规模发电通常**分布式嵌入**电网，包括家庭级别
- 随着"净零排放"和《巴黎协定》等倡议的推进，可再生能源采用率将持续增长

### 1.2 研究问题
现代电网正在经历拓扑结构、组成和动态特性的全面变革：
- 网络从传统的**集中式**（少量高功率发电机）转向**高度分布式**
- 双向电力流动（家庭既是消费者也可能是生产者）
- 发电机随时上线/离线，网络结构可变

**核心问题**：这种日益分布式的布局如何影响电网的稳定性和韧性？

## 二、研究方法

### 2.1 动力学模型：摇摆方程（Swing Equation）

研究采用摇摆方程来描述电网节点相位角θ相对于电网参考频率Ω的偏差：

$$\frac{d^2\theta_i}{dt^2} + \gamma\frac{d\theta_i}{dt} = P_i - \kappa\sum_{j=1}^{n} A_{ij}\sin(\theta_i - \theta_j)$$

**参数说明**：
| 参数 | 含义 | 说明 |
|------|------|------|
| θᵢ | 节点i的相位角偏差 | 表示与参考频率的偏离 |
| γ | 阻尼系数 | 表征能量耗散 |
| κ | 耦合容量 | 网络传输能力 |
| Pᵢ | 节点功率 | 正值为发电，负值为消耗 |
| Aᵢⱼ | 邻接矩阵 | 描述网络拓扑结构 |

**物理意义**：方程表达了每个节点的惯性、耗散和传输电功率之间的平衡。

### 2.2 网络拓扑模型

采用**Watts-Strogatz网络**模型，通过随机重连参数q ∈ [0,1]控制网络结构：
- q = 0：规则格点网络
- q = 0.1：小世界网络
- q = 1：泊松随机网络

### 2.3 数据来源

| 数据类型 | 来源 | 规模 | 分辨率 |
|----------|------|------|--------|
| 家庭用电消耗 | 英国电力网络 | 约5000户伦敦家庭 | 30分钟 |
| 光伏发电量 | 英国电力网络 | 100块独立光伏板 | 10分钟 |
| 时间跨度 | - | 2年（消耗）/1年（发电） | - |

## 三、核心概念

### 3.1 临界耦合容量（κc）
- **定义**：使电网保持同步运行所需的最小耦合强度
- **意义**：κc越低，电网越容易同步，越不需要额外容量
- **计算方法**：通过数值积分方程(1)找到稳态固定点

### 3.2 韧性度量（ρ）
- **定义**：ρ = αc/α*，其中αc是级联失效后半数边存活所需的临界边容量，α*是正常运行所需的最小边容量
- **意义**：ρ越低，韧性越高
- **分布特征**：ρ在网络集成中服从**对数正态分布**

### 3.3 节点类型
| 类型 | 符号 | 定义 |
|------|------|------|
| 发电机 | n₊ | Pᵢ > 0，净发电 |
| 消费者 | n₋ | Pᵢ < 0，净消耗 |
| 被动节点 | nₚ | Pᵢ ≈ 0，净功率接近零 |

## 四、主要发现

### 4.1 电网耦合与稳定性

**发现1：最优配置位于单形中心**

通过将网络配置投影到(n₊, n₋, nₚ)构成的三角单形上分析发现：

- 当**发电机数量与消费者数量相当**时，临界耦合κc最小
- 被动节点较少时，电网更容易同步
- 随着拓扑随机性q增加，κc整体降低（更随机的网络更易同步）

**示例**：
- 规则格点（q=0）只有一个消费者时，κc约为最大功率的1/2
- 小世界网络（q=0.1）相同配置下，κc仅为最大功率的1/4

### 4.2 级联失效机制

**失效类型**：
1. **过载失效**：边的功率流超过最大容量α
2. **去同步失效**：边上相位差超过1Hz安全阈值

**发现2：灾难性级联发生极快**

| 级联严重程度 | 持续时间 | 主要失效原因 |
|--------------|----------|--------------|
| 灾难性（完全失效） | 极短（< 数秒） | 主要为过载 |
| 中等损害 | 20-30秒峰值 | 过载与去同步各半 |
| 轻微扰动 | 较长，趋于稳定 | 小幅波动 |

**重要启示**：对最具破坏性级联的对策必须是**预防性**的，而非反应性的。

### 4.3 韧性景观分析

**发现3：最韧性区域在单形中心底部**

| 单形位置 | 韧性水平 | 原因 |
|----------|----------|------|
| 中心底部 | 最高（ρ最低） | 发电机≈消费者，被动节点少 |
| 边缘区域 | 最低（ρ最高） | 发电机与消费者严重不平衡 |

随着网络随机性q增加：
- ρ整体升高
- 低韧性区域从边缘向内部扩展
- 但最韧性区域始终在中心底部

### 4.4 微电网的日变化轨迹

**发现4：微电网运行轨迹主要位于低韧性区域**

通过定义功率生成/消耗密度(η₊, η₋, ηₚ)，将实际数据映射到单形上：

**日变化特征**：
- **午夜**：无发电但有消耗 → 位于单形右上角（多消费者、多被动）
- **正午**：大量发电、少量消费 → 位于单形左侧（多发电机）
- **整体**：在单形两侧**摆动**，仅短暂经过中心区域

**季节与PV采用率影响**：

| 季节 | PV采用率 | 轨迹特征 |
|------|----------|----------|
| 冬季 | 50% | 局限于单形右侧小区域 |
| 冬季 | 100% | 可穿越单形但范围有限 |
| 夏季 | 50% | 可穿越单形 |
| 夏季 | 100% | 大幅向左侧延伸（发电主导） |

### 4.5 临界耦合的日振荡

**发现5：κc呈现明显的日周期振荡**

- 大部分时间保持相对恒定
- 早晨出现小峰值
- 当轨迹穿越单形中心时出现**短暂谷值**（最佳同步时机）
- 这些谷值持续时间极短，因为发电高峰与消费高峰存在时间差

### 4.6 PV高采用率的韧性代价

**发现6：高PV采用率显著降低夏季电网韧性**

| PV采用率 | 季节 | αc分布 | 原因 |
|----------|------|--------|------|
| 50% | 夏季 | 单峰，较低值 | 适度分布 |
| 100% | 夏季 | **双峰**，出现高值峰 | 白天高功率输出集中 |

双峰分布解释：
- **较小峰**：晚间失效，需求异质但功率较低
- **较大峰**：白天高PV输出时失效，需要5-15 kW临界容量

**关键警示**：低压配电网线路额定值通常为4-15 kW，高PV采用率的微电网在夏季**接近临界运行状态**。

## 五、电池储能的影响

### 5.1 电池模型
基于**Tesla Powerwall 2**建模：
- 最大充放电率：5 kW
- 最大存储容量：约14 kWh
- 运行策略：仅优化单个家庭自给自足

### 5.2 关键发现

**发现7：传统家用电池对电网韧性改善有限**

| 方面 | 无电池 | 有电池 | 评价 |
|------|--------|--------|------|
| 家庭自给自足 | 低 | **大幅提高** | ✓ 积极 |
| 被动节点比例 | 较低 | **增加** | ⚠ 中性 |
| 轨迹位置 | 单形边缘 | **仍在边缘**（尤其左侧） | ✗ 问题 |
| κc日振荡 | 存在 | 振荡**保持甚至加剧** | ✗ 问题 |
| αc双峰性 | 存在 | **保持** | ✗ 问题 |

**原因**：当前电池设计仅优化个体自给自足，不考虑网络整体韧性。电池使家庭变得更"被动"，但这恰恰将轨迹推向**韧性最差的边缘区域**。

## 六、补充材料要点

### 6.1 摇摆方程的详细推导
每个节点建模为旋转机器，相位角φᵢ(t) = Ωt + θᵢ(t)

功率平衡要求：
- 惯性功率：$P_i^{iner} = \frac{1}{2}J_i\frac{d}{dt}\left(\frac{d\phi_i}{dt}\right)^2$
- 耗散功率：$P_i^{diss} = D_i\left(\frac{d\phi_i}{dt}\right)^2$
- 传输电功率：$P_i^{elec} = \sum_{j=1}^{n} V_0^2 B_{ij}\sin(\theta_i - \theta_j)$

### 6.2 双节点系统的稳定性分析
简化为一个发电机（功率P）和一个消费者（功率-P）：

- 存在两个固定点：
  - 稳定：Δθ = arcsin(P/κ)
  - 鞍点：Δθ = π - arcsin(P/κ)
- 在κc = P处发生**鞍结分岔**，两固定点湮灭
- 临界耦合的物理量级估算：V₀²B ≈ 18.3 kW（V₀=240V，L=10mH）

### 6.3 级联持续时间的系统分析
- 灾难性级联（低α）：持续时间极短
- 随α增加，持续时间上升至αc/α*处达到峰值（20-30秒）
- 继续增加α，持续时间下降并趋于稳定

### 6.4 不同网络结构的韧性对比
- q增加 → ρ整体升高 → 边缘低韧性区向内扩展
- 但所有q值下，最韧性配置始终在单形中心底部
- n=50与n=100的网络表现出相同的单形形态

## 七、研究意义与展望

### 7.1 核心结论

1. **分布式不等于韧性**：虽然可再生能源在空间上分布，但其输出在时间上**高度聚集**，导致电网处理大功率流，对灾难性失效脆弱。

2. **供需时间错配**：微电网大部分时间运行在韧性景观的**最不利区域**，仅短暂经过最优区域。

3. **电池的局限性**：当前家用电池设计**无法改善**这一问题，因为它们仅优化个体而非系统。

### 7.2 对未来电网设计的启示

| 挑战 | 当前状态 | 期望方向 |
|------|----------|----------|
| 轨迹位置 | 边缘区域 | 推向中心底部 |
| 电池策略 | 个体自给自足 | **系统级协调调度** |
| 功率时间分布 | 高度集群 | 时间平滑 |

### 7.3 潜在解决方案

1. **新型电池调度策略**：不仅优化个体自给自足，还能向上游网络回馈能量

2. **车辆到电网（V2G）技术**：利用电动汽车作为分布式储能资源

3. **虚拟电厂**：整合储能和能源管理控制系统

4. **动态网络感知控制**：未来控制方案必须考虑网络动力学特性
